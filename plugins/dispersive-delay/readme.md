# dispersive-delay

dispersive-delay is the rework version of [sdelay](https://github.com/a5632645/sdelay)

> [!NOTE]  
> This plugin requires AVX or AVX2 support.  

> [!WARNING]  
> massive cpu consume when too much filters.
